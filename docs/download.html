Conrad is licensed under the Lesser Gnu Public License (LGPL). Two distributions are available for Conrad:

The main distribution contains everything you need to run Conrad and even customize it for your own models.

Download conradV1.zip

The source distribution contains everything in the main distribution plus the full source code and unit tests for Conrad packaged as an Eclipse project. Although not strictly necessary, you will probably want this as a reference if you plan to write your own features or I/O handling.

Download conradSrcV1.zip

Previous versions and more info are available on Sourceforge at http://sourceforge.net/projects/conradcrf.
